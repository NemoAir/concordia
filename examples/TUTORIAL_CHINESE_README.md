# Concordia 中文教程：鸿门宴模拟

## 📖 简介

这是一个基于 Concordia 框架的中文历史对话模拟教程，以著名的"鸿门宴"历史事件为背景。

## 🎭 场景

**时间**：公元前 206 年
**人物**：项羽（西楚霸王） vs 刘邦（汉王）

## 🚀 快速开始

1. 设置 API 密钥（Google AI Studio）
2. 运行 `tutorial_chinese.ipynb`
3. 观察项羽和刘邦的 AI 对话

## 📊 特点

- ✅ 完全中文化
- ✅ Token 限制已优化
- ✅ 详细的历史背景
- ✅ 易于扩展到其他场景

## 🎯 学习要点

1. 中文提示词优化
2. 角色目标设定
3. 历史背景配置
4. 多轮对话模拟
